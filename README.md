# Mô tả game
## Cách chơi
- Nhấn chuột **trái** để mở ô.
- Nhấn chuột **phải** để cắm cờ.
- **Điều kiện Thắng** mở tất cả các ô mà không phải ô chứa bom
## Thành phần game
- gồm menu, chọn độ khó và giao diện trò chơi
- Nhấn escape hoặc nút X trên cửa sổ góc phải để thoát
